# 系统总览

## 产品定位

applecry 公开工作台是一个无后端构建步骤的静态网站，用来公开展示个人创作、AI 产品实验、写作、每日信息信号和展览发现工具。它不是通用搜索引擎、票务交易平台或带账号体系的 SaaS。

## 核心模块

- 公开工作台：首页展示个人定位、近期记录、笔记、AI/美股/A股新闻、夜间歌单、项目和近况。
- Exhibit Atlas：用地图与列表发现全球展览，支持地区、类别、日期、搜索、排序、收藏、详情和 ICS 日历。
- 漫展嘉宾雷达：从已核验嘉宾出发发现近期漫展，支持搜索、范围筛选、排序、嘉宾面板和票务/官宣来源。
- Voice PageAgent：网页内的操作型助手，支持文字任务、语音转写、跨站内页面会话续跑和页面操作。

## 技术边界

- 页面以 HTML、CSS 和原生 JavaScript 为主，部署到 Cloudflare Pages。
- 新闻、展会、漫展和社交线索来自仓库内 JSON 文件。
- 主题、收藏和 PageAgent 会话保存在浏览器存储中，不代表云端同步。
- PageAgent 使用本地 SDK，并在本地脚本不可用时尝试 CDN；模型请求由配置的模型网关处理。
- PageAgent 面板和唤醒按钮标有 `data-page-agent-ignore`，不会被当作业务页面元素重复操作。

## PageAgent 判断协议

1. 先从知识库确认页面目标与数据边界。
2. 再读取当前 URL、可见内容、控件状态和加载提示。
3. 操作后验证状态变化；失败时按排障顺序缩小问题。
4. 输出时区分稳定规则、当前观察和推断。
5. 不把小红书线索当成展会事实，不把票务海报人物当成正式嘉宾。
